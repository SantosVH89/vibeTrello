# Estructura de Carpetas del Proyecto

## .agent
**Descripción:**  
Donde se guardan las *Skills* de Antigravity.

---

## .git
**Descripción:**  
Gestión de versiones con Git.

**Incluye:**
- Historial de versiones del proyecto
- Configuración del repositorio
- Archivo `.gitignore`

---

## .ide
**Descripción:**  
Donde se guardan las *Skills* de Windsurf.

---

## backend
**Descripción:**  
Carpeta donde se guarda todo lo relacionado con el **Backend**.

**Ejemplos:**
- APIs
- Lógica de negocio
- Conexión con base de datos
- Servicios

---

## analisis / desarrollo
**Descripción:**  
Carpeta destinada al **análisis y desarrollo del proyecto**.

**Incluye:**
- Documentación del ciclo de vida del software
- Diseños de arquitectura
- Planificación
- Implementaciones que no pertenecen directamente a **backend** ni **frontend**

---

## frontend
**Descripción:**  
Carpeta donde se guarda todo lo relacionado con el **Frontend**.

**Ejemplos:**
- Interfaces de usuario
- Componentes visuales
- Estilos
- Interacción con el usuario
